
# A-Spotify-clone.

A-Spotify-clone.


## Features

Modern homepage, fully-fledged music player, search, lyrics, song exploration features, search, popular music around you, worldwide top charts, and much more



    
    
    
    
    

## Tech Stack

HTML, CSS, JS, Reactjs, Redux, Tailwind, RapidAPI
